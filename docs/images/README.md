# Screenshots

Placeholder assets for the README. Replace these SVGs with real PNG captures when preparing a portfolio release.

## Recommended captures

| File | What to capture | Suggested viewport |
|------|-----------------|-------------------|
| `dashboard-manager.png` | Manager **Team Dashboard** with survey stats | 1440×900 |
| `templates.png` | **Survey Templates** list with at least one template | 1440×900 |
| `results-charts.png` | **Team Results** detail page with radar/bar charts | 1440×900 |
| `admin-users.png` | Admin **User Management** grid | 1440×900 |
| `login.png` | Login screen (no secrets in frame) | 1280×800 |

## How to capture locally

1. Start the stack: `docker compose up --build`
2. Open http://localhost:3000
3. Sign in with seed accounts (see root README)
4. Use browser DevTools → **Capture screenshot** (full viewport) or a tool like ShareX
5. Save PNGs in this folder using the filenames above
6. Update README image references from `.svg` to `.png` when ready
